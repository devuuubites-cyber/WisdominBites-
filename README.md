# WisdomInBites — Wisdom in Minutes

A minimalist, high-performance web platform designed to teach powerful intellectual frameworks (Mental Models and Cognitive Biases) in short, digestible formats.

## 🚀 Live Demo
[View the Live Site on GitHub Pages](https://your-username.github.io/your-repo-name/)

## ✨ Key Features
* **Immersive Design:** Obsidian-themed UI with a custom starfield background and radial glow effects.
* **Interactive Learning:** Uses an Intersection Observer API for smooth "scroll-to-reveal" animations.
* **Custom Experience:** Features a custom gold-cursor tracking system and a reading progress bar.
* **Responsive:** Fully optimized for mobile, tablet, and desktop viewing.

## 🛠️ Built With
* **HTML5 & CSS3:** Semantic structure and advanced CSS variables/animations.
* **JavaScript (ES6):** Intersection Observer for scroll effects and Canvas API for the starfield background.
* **Typography:** Playfair Display for elegance and DM Mono for technical precision.

## 📖 What's Inside?
WisdomInBites focuses on two core pillars of intelligent thinking:
1.  **Cognitive Biases:** Understanding predictable errors in human judgment (e.g., Confirmation Bias, Loss Aversion).
2.  **Mental Models:** Tools for structured problem solving (e.g., First Principles, Inversion).
